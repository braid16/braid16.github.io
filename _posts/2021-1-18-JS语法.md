## &&短路逻辑

A&&B&&C&&D 取第一个假值或D ，并不会取 true / false

## || 短路逻辑

A||B||C||D 取第一个真值或D，并不会取 true / false

## while 循环

语法

while(表达式) {语句}

判断表达式的真假

当表达式为真，执行语句，执行完再判断表达式的真假

当表达式为假，执行后面的语句

其他

do...while 用得不多，

## for循环

语法糖

for式while循环的方便写法

语法

for(语句1;表达式2;语句3) {

   循环体

}

先执行语句1

然后判断表达式2

如果为真，执行循环体，然后执行语句3

如果为假，直接退出循环，执行后面的语句

## break和continue

退出所有循环 和 退出当前一次循环的差别。
